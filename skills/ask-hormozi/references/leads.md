# Leads — Creating Qualified Demand

Use this guide when too few right-fit prospects enter the sales process. It is a paraphrased operating guide. Retrieve attribution from `references/evidence/leads.md`.

## Define the output

A contact is reachable. An engaged lead has shown interest through a reply, opt-in, click, application, booking, or equivalent action. Optimize for qualified engaged leads, not list size or impressions.

Track:

- engaged leads by source;
- qualified opportunities;
- cost per qualified opportunity;
- booked and attended calls;
- customers and fully loaded CAC.

## Choose one acquisition path

The Core Four organize acquisition by relationship and reach:

| | One-to-one | One-to-many |
|---|---|---|
| Existing relationship | Warm outreach | Free content |
| No existing relationship | Cold outreach | Paid advertising |

Choose one primary path for a defined test window. Add another only after the current path has enough volume and measurement to judge.

- **Warm outreach:** Start with people who know the operator. Personalize, ask about them, test interest, and request introductions.
- **Free content:** Earn attention with a strong topic, hold it with useful structure, deliver the promised value, then use a clear next step.
- **Cold outreach:** Narrow the list, personalize the reason for contact, make one relevant offer, and follow up consistently.
- **Paid advertising:** Scale only when tracking, conversion, fulfillment, and payback are stable.

## Lead magnet

Create a complete solution to a narrow problem that attracts the intended buyer and naturally reveals the next problem the paid offer solves.

1. Pick one painful problem and one buyer.
2. Choose diagnosis, sample, or one useful step as the mechanism.
3. Package it in the easiest format to consume.
4. Test the title and promise.
5. Deliver a real quick win.
6. Give one explicit call to action.

## Volume and iteration

The Rule of 100 is a volume heuristic: commit to a meaningful daily input on one path long enough to produce evidence. Adapt the unit to the business and channel; do not prescribe blind outreach where consent, regulation, reputation, or platform rules make it unsafe.

Run this loop:

1. Set the weekly input.
2. Measure engaged and qualified output.
3. Inspect message, list, offer, and channel separately.
4. Improve one variable.
5. Repeat before adding a new channel.

## Completion check

A lead prescription is complete when it names one path, one weekly input, one qualification measure, and the capacity limit that would stop further volume.
