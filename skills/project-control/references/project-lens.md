# Project lens

The lens is a set of questions, not a schema. Answer only what the project's sources support, using the project's own vocabulary.

## Universal questions

1. **Outcome:** What is this project trying to make true?
2. **Current reality:** What has actually happened, what is happening now, and what is waiting?
3. **Structure:** Which relationships matter here—sequence, dependency, ownership, handoff, evidence, decision, or another project-specific relationship?
4. **Pressure:** What is blocked, uncertain, contradicted, overdue, or awaiting a person or event?
5. **Implication:** Why does the current situation matter to the outcome?
6. **Next moves:** What can happen next, and which items are facts versus recommendations?
7. **Freshness:** Which sources support the view, and when were they observed?

## Evidence language

Label important statements as:

- **Observed:** directly supported by a file, command, test, tracker, or connected source.
- **Reported:** stated by the user or a named person but not independently verified here.
- **Inferred:** reasoned from supported facts; show the reasoning.
- **Hypothetical:** introduced for scenario exploration.
- **Unknown:** materially unanswered by the available sources.

Absence of evidence remains unknown. A list's order does not establish dependency. A status label does not establish proof.

## Dynamic vocabulary

Use whatever entities the project actually contains: tasks, bets, scenes, clients, experiments, approvals, claims, systems, decisions, milestones, assets, risks, or something else. Preserve their native names rather than translating them into a universal taxonomy.

The lens may be held in working context or represented as temporary inline data in the visualization. A serialized snapshot is optional and has no required shape beyond remaining traceable to its sources.

## Plain-English handoff

Before rendering, be able to say:

> The project is trying to ____. Right now ____. The main thing affecting progress is ____. That matters because ____. The next available moves are ____.

If the sources cannot support that statement, name the missing information instead of creating a decorative dashboard.
